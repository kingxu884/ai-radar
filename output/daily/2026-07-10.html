🔬 值得深挖的论文/报告
DeepSearch-World: Self-Distillation for Deep Search Agents in a Verifiable Environment
核心贡献：构建了一个包含 42 万条多跳 QA 任务的可验证环境，Agent 能在其中进行自我蒸馏学习——包括进度验证、有根据的反思和失败恢复。这解决了 Agent 训练中"稀疏奖励"的核心难题。
为什么值得读：如果你正在构建需要多步推理和工具调用的 Agent 系统，这篇论文提供了可复现的训练框架。可验证环境的设计思路可以直接迁移到企业级 Agent 的质量保障体系。
Hidden Decoding at Scale: Latent Computation Scaling for Large Language Models
核心贡献：提出 Hidden Decoding 方法，通过在序列长度维度增加计算量来扩展模型能力，而非扩大 Transformer 骨干网络。额外计算只是更长的输入，与标准大模型训练兼容。
为什么值得读：这提供了一种"不重新预训练就能让现有模型更强"的可能路径。论文讨论的"深度循环 Transformer 难以规模化"问题是当前架构研究的核心痛点，该方法绕开了这个难题。
DrugGen 2: A disease-aware language model for enhancing drug discovery
核心贡献：以疾病本体和靶蛋白序列为条件生成小分子药物，采用 SFT + GRPO 两阶段训练策略，优化类药性和合成可达性。
为什么值得读：将疾病上下文纳入药物生成是 AI 药物发现的关键进化方向。论文的 RL 调优策略（GRPO）与 LLM 对齐技术同源，代表了 AI for Science 与 LLM 方法论的深度融合趋势。
