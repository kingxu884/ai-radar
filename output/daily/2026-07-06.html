发生了什么：开发者社区正迅速收敛共识，将 MCP（模型上下文协议）定义为“给 Agent 双手（连接工具与数据）”，A2A（Agent-to-Agent 协议）定义为“给 Agent 同事（多 Agent 协作）”，加上 WebMCP 构成三层标准架构。
