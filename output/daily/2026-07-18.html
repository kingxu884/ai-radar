📈 趋势分析

趋势一：Harness 工程成为 Agent 时代的新基建

证据链：DeepSeek 单独设立 Harness 部门并紧急扩招（Harness 研究员、工程师、产品经理）→ 面壁智能开源 StaffDeck 框架（AI 身份管理、权限控制、绩效评估）→ 行业研究指出“Harness 优化比微调能带来 13.70 基准点提升” → LangChain 通过改进 harness 将编码 agent 在 Terminal Bench 2.0 上从 52.8% 提升至 66.5%。

预判：2026 年下半年，Harness 工程将取代 Prompt 工程成为 AI 工程师最核心的技能。MCP（模型上下文协议）、A2A（Agent 间通信协议）、Ralph Loop（自修正循环）、Hard Sandboxing（强隔离执行）将成为技术选型的关键词。企业招聘 AI 工程师时，“熟练掌握 MCP 协议”可能成为新的硬性要求。

趋势二：模型能力商品化加速，竞争焦点转向“最后一公里”落地

证据链：Anthropic 被迫将 Fable 5 纳入订阅计划（竞争压力）→ Databricks 用开源模型经济学挑战闭源方案 → 印度企业因成本优势选择 DeepSeek 等中国模型 → 智谱 ARR 半年增长 15 倍（说明企业开始为“端到端方案”而非“模型能力”买单）。

预判：基础模型本身正在成为“大宗商品”，利润将向两端转移：一是掌握芯片和数据中心的“算力层”（英伟达、算力租赁商），二是掌握垂直场景和用户关系的“应用层”（企业级 Agent 平台）。纯粹的模型层公司如果不向上或向下延伸，将面临严重的利润挤压。

趋势三：AI 基础设施投资从“训练”转向“推理”

证据链：TechCrunch 报道“第一批 GPU 金融家转向推理芯片，达成 4 亿美元交易” → 国家能源局指出“十五五时期推理负荷将超过训练负荷” → PPIO 发布面向 Agent 时代的智能 Token 工厂，万亿 Token 调用量验证 → 清华系 Token 优化工厂兼容 10 余种国产芯片。

预判：推理基础设施将成为下一个资本密集投入领域。与训练阶段的“集中式、高算力”需求不同，推理要求“低延迟、广覆盖、高并发”，这将催生边缘推理节点、推理芯片定制、跨云推理调度等新赛道。国产芯片厂商如果能抓住“推理优化”这一差异化需求，可能找到突破英伟达垄断的窗口。
