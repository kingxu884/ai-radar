🔬 值得深挖的论文/报告
1. Token Reduction Is Not Cost Reduction (arXiv:2607.12161) 核心贡献：对 Claude Code 在 103 个任务、7 个代码库上做了 2,848 次付费执行的双变量对照实验，证明上下文压缩不必然降低实际付费成本。揭示了上下文窗口的经济学远比“少 token 就省钱”复杂。为什么值得读：每一个在优化 Agent 成本的人都需要看这篇——它会颠覆你对 prompt 压缩、检索排序和 payload 优化的假设。对你的代码库做同样的计费对照实验，可能发现完全不同的优化方向。
2. A Shared Subcircuit Lets LLMs Count Down Across Tasks (arXiv:2607.12279) 核心贡献：在 Llama-3.1-70B-Instruct 中发现了“倒计时子回路”——一个跨任务的通用机制，让模型能追踪距离目标还有多少 token。该回路使用的几何模式与之前发现的其他通用模式一致。为什么值得读：如果你做可解释性或希望理解 LLM 的推理机制，这篇展示了“能力是如何在回路级别复用的”。对 Agent 的规划能力研究尤其重要，因为规划需要精确的距离估计。
3. Agentic Systems for Breast Cancer Treatment Recommendations (arXiv:2607.12051) 核心贡献：在 72 个真实乳腺癌病例上对比了 7 种 Agent 架构，发现多 Agent 架构（含事实核查和自主子 Agent 生成）显著优于单模型设定。最佳配置达到临床可接受水平。为什么值得读：这是 Multi-Agent 在高度专业化、高风险的医疗场景中跑通闭环的罕见案例。其架构选择（监督者-子 Agent 生成-事实核查）可以为其他垂直领域（法律、财务）提供直接参考。
