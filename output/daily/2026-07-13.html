🔬 值得深挖的论文/报告
1. KV-PRM: Efficient Process Reward Modeling via KV-Cache Transfer for Multi-Agent Test-Time Scaling 核心贡献： 用 LLM 生成时自然产生的 KV cache 替代重编码文本，把 PRM 计算的二次方复杂度降下来，直接打通了 Multi-Agent 长程推理在工程上规模化部署的瓶颈。 为什么值得读： 它是 Agent 效率问题的一个"优雅偷懒"——不做加法（不访问外部模块）而是"就地取材"。掌握这项技术可以显著降低你 Agent 框架的响应延迟和计算开销。
2. GATS: Graph-Augmented Tree Search with Layered World Models for Efficient Agent Planning 核心贡献： 规划阶段完全消除 LLM 调用，用三层世界模型（符号匹配+统计+LLM 兜底） + UCB1 树搜索产出更优规划，LLM 只在离线构建世界模型时使用。 为什么值得读： 重新定义 LLM 与经典规划算法的分工边界。如果你的 Agent 面对重复性、规则明确的任务，这类架构可能比盲目堆 LLM 调用的方案更稳定、更经济。
3. Toward Auditable AI Scientists: A Hypothesis Evolution Protocol for LLM Agents 核心贡献： 在材料科学场景下实现假设生成-测试-修正的可审计闭环，非黑箱输出，而是结构化的科学推理日志。 为什么值得读： AI4S 落地最根本的阻碍不是模型不够强，而是科学家不敢信。这份工作给出了一套在"人机协作科学发现"中建立信任的技术协议，是所有做 AI 辅助科研产品的人应该关注的范式。
